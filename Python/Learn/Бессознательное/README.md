# Бессознательное

Папка с учебными и экспериментальными Python-скриптами для консольного диалога с "бессознательным". Здесь собраны несколько версий одной идеи: от простого ответа "Да/Нет" до тематических ответов по ключевым словам.

## Что где лежит

- `Dear_unconscious_1.0.py` - первая версия сценария: проверка готовности и простой диалог.
- `Dear_unconscious_1.0 copy.py` - более аккуратная консольная версия с функциями и циклом сессии.
- `dear unconscious_2.0.py` - версия 2.0 с циклом вопросов и ожиданием между попытками.
- `dear unconscious_3.0.py` - версия 3.0 с медитативной подготовкой перед вопросами.
- `dear unconscious_Mif_card.py` - версия с тематическими ответами по ключевым словам в вопросе.
- `__pycache__/` - сгенерированные Python-файлы после запуска скриптов.

## Описание файлов

- `Dear_unconscious_1.0.py` - содержит простую проверку готовности и демонстрацию диалога; внутри есть две части: базовая версия 1.0 и цикл версии 1.5.
- `Dear_unconscious_1.0 copy.py` - вынесены отдельные функции для готовности, ответа и цикла сессии; есть более удобный выход из программы.
- `dear unconscious_2.0.py` - версия с бесконечным циклом, случайной готовностью и задержкой перед следующей попыткой.
- `dear unconscious_3.0.py` - расширяет 2.0 добавлением функции `preparation()` с паузами и текстом для настройки перед сеансом.
- `dear unconscious_Mif_card.py` - анализирует текст вопроса по ключевым словам и возвращает тематические ответы: тревога, работа, деньги, отношения, здоровье и т.д.

## Примечание

Имена файлов частично экспериментальные, поэтому в папке есть несколько близких по смыслу версий одной и той же идеи.
