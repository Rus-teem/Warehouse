# Интерактивный режим Python — это режим работы, при котором код выполняется сразу после ввода команд. Это удобно для быстрого тестирования идей, проверки функций или
экспериментов с библиотеками.

Команда import в Python используется для подключения модулей или библиотек, чтобы использовать их функции, классы, переменные или другие объекты. 
Это позволяет использовать уже готовый код, не переписывая его заново.

requests - модуль 

Модуль/библиотекка  requests - Простая в использовании HTTP-библиотека для Python Модуль requests - это элегантная и простая HTTP-библиотека для Python, созданная для людей.
Модуль позволяет чрезвычайно легко отправлять HTTP/1.1 запросы. Нет необходимости вручную составлять URL-адреса или кодировать данные для PUT и POST запросов (в настоящее 
время можно использовать метод Request.json)

BeautifulSoup - модель 

BeautifulSoup – это библиотека для парсинга HTML и XML документов в Python. Она предоставляет простой и удобный способ извлекать данные из веб-страниц, а также 
облегчает работу с этими данными.
